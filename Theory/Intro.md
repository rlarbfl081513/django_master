# 앞으로 알아둘 것
1. 원형 : serve 
2. er : server  ~~하는 기계나 사람
3. ing : serving ~~하는 행위
4. 예시
    1. 예시 1번
        1. program : 순서도(무한도전 18시)
        2. programmer : 순서도를 작성하는 사람이나 기계
        3. programing : 순서도를 작성하는 행위
    2. 예시 2번
        1. route  : 경로
        2. router : 경로를 찾는 기계나 사람
        3. routing : 경로를 찾는 행위
